The Crime Intelligence and Prediction System is an end-to-end data analytics and machine learning web application developed using Python and Streamlit to analyze crime patterns across Indian cities and generate actionable insights. The platform transforms raw crime incident records into meaningful intelligence through automated processing, visualization, prediction, forecasting, and reporting modules. It is designed to support data-driven decision making for researchers, analysts, and public safety planning by combining statistical analysis with modern machine learning techniques.

The application begins with an automated data cleaning pipeline that processes uploaded crime datasets to ensure reliability and consistency. Crime occurrence timestamps are standardized into structured datetime formats, invalid or missing records are removed, and temporal attributes such as year and month are extracted. The system aggregates incidents into monthly crime counts per city, converting unstructured records into a structured analytical dataset suitable for machine learning modeling and time series analysis. This preprocessing stage ensures accurate insights and stable predictive performance.

The interactive dashboard provides a comprehensive analytical overview of crime trends through dynamic visualizations and key performance indicators. Users can explore yearly crime growth patterns, monthly distribution trends, seasonal variations, and city-wise crime contributions. Summary metrics such as total reported crimes, peak crime year, and highest crime contributing city offer quick intelligence insights. These analytics enable users to understand historical crime behavior and identify patterns across locations and time periods.

For predictive intelligence, the system integrates a Random Forest regression model that estimates future crime counts based on selected city and temporal parameters. The model captures complex relationships between location and time to produce reliable predictions. In addition to machine learning prediction, the platform includes ARIMA-based time series forecasting, allowing users to analyze historical crime trends and forecast future crime patterns for extended periods. This combination supports both short-term prediction and long-term planning scenarios.

The application also provides city comparison analytics that allow users to evaluate crime trends between different cities across multiple years, helping identify regional differences and growth patterns. Built with a modern Streamlit interface, the system offers intuitive multi-page navigation, responsive layouts, and interactive controls for seamless exploration. To support documentation and sharing of insights, the platform includes automated PDF report generation that compiles analytical visualizations and prediction summaries into structured intelligence reports suitable for presentations and research purposes.
